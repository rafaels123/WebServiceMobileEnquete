WebServiceMobileEleicao
=======================

Trabalho de conclusão de curso.

Aplicação de uma enquete para os projetos que
serão apresentados na semana de informática da FATENE,
para a aplicação será criado um webservice restful que
será consumido por qualquer cliente independente de plataforma.

Tecnologias utilizadas
 - Spring
 - Hibernate
 - maven
 - jax-rs
